YouTube Lecture Summarizer

This extension helps you summarize YouTube lecture videos that have captions/transcripts.

Error: There was a problem connecting to the YouTube page. Please try refreshing the page and clicking the extension icon again.
1
Navigate to a YouTube video that has captions/transcripts (typically educational videos, lectures, TED talks, etc.)
2
Once on a YouTube video page, click the extension icon again
3
A summary panel will appear on the right side of the video
4
Click "Summarize" to generate a summary of the lecture
5
You can copy the summary or download it as a text file

Note: This extension only works with YouTube videos that have captions/transcripts.

Troubleshooting Tips:

  • Make sure you're on a YouTube video page (URL should include youtube.com/watch)
  • Refresh the page if the extension doesn't work the first time
  • Check if the video has captions by clicking the CC button in the YouTube player
  • Some videos may have auto-generated captions which might not be as accurate
  • If you see an error message, try reloading the extension from the Chrome extensions page
